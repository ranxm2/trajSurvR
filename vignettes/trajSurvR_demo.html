trajSurvR Demo

Overview

This vignette demonstrates the core workflow wrapped from the raw/*.Rmd analyses:

  1. QC
  2. Differential analysis
  3. Correlation study
  4. Trajectory modeling
  5. Time-to-event analysis (baseline Cox + LASSO Cox + time-dependent Cox)
  6. Event prediction (logistic regression)
  7. Time-of-onset prediction (regression with baseline)
  8. Feature selection (LASSO + forward greedy)

class_counts <- table(pred_dat$CASE)
if (length(class_counts) == 2 && all(class_counts >= 2)) {
  lasso_event <- fit_event_prediction_lasso(
    data = pred_dat,
    outcome_col = "CASE",
    feature_cols = c("delta_001", "delta_002", "delta_003", "CollAge", "Weight", "Height"),
    nfolds = 3
  )
  lasso_event$selected_min
} else {
  cat("Example dataset has too few cases/controls for logistic LASSO; skipping.\n")
